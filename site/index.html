@thaibulksms/tbs
GitHub npm API Docs

SMS & Email from your terminal
and AI agents

Zero-dependency MCP server & CLI for ThaiBulkSMS and ThaiBulkMail. Works with Claude Code, Gemini CLI, OpenCode, and OpenClaw.

0 dependencies Node.js 18+ Bun 1.3+ 8 MCP tools MIT
terminal
$ npm install -g @thaibulksms/tbs copy

CLI Commands

tbs login

Save API credentials to ~/.config/tbs/ with profile support

tbs send <to> <msg>

Send SMS to Thai mobile number with custom sender ID

tbs email <to> <subj>

Send email via ThaiBulkMail template

tbs credit

Check SMS and email credit balance

tbs otp <to>

Request OTP via SMS, returns verification token

tbs email-otp <to>

Request OTP via email template

Add to AI Agent

Claude Code
claude mcp add thaibulksms \
  -e THAIBULKSMS_API_KEY=xxx \
  -e THAIBULKSMS_API_SECRET=xxx \
  -e THAIBULKSMS_OTP_KEY=xxx \
  -e THAIBULKSMS_OTP_SECRET=xxx \
  -- npx -y @thaibulksms/tbs
Gemini CLI
gemini mcp add thaibulksms -- npx -y @thaibulksms/tbs
OpenCode (opencode.json)
{
  "mcp": {
    "thaibulksms": {
      "type": "local",
      "command": ["npx", "-y", "@thaibulksms/tbs"],
      "environment": {
        "THAIBULKSMS_API_KEY": "{env:THAIBULKSMS_API_KEY}",
        "THAIBULKSMS_API_SECRET": "{env:THAIBULKSMS_API_SECRET}"
      },
      "enabled": true
    }
  }
}
OpenClaw
Uses .mcp.json (same as Claude Code) — drop in project root.
Claude Desktop
{
  "mcpServers": {
